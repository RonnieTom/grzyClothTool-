**grzyClothTool** 是一款免费工具，用于轻松创建和管理 GTA5 附加包服装。  
现在你可以完成**几乎**所有其他同类工具能做的事，而且**完全免费**！

##

# 为什么选择 grzyClothTool？

- 独家功能
  - 轻松预览纹理
  - 轻松预览所有 Ped 配饰
  - 可同时预览多个 Drawable
  - 在 3D 预览中查看头发在帽子下的收缩效果
  - 在 3D 预览中查看高跟鞋所需的高度补偿
  - 无需担心单个附加包 128 件物品上限，工具会自动拆分为多个附加包！
- 开源
  - 无需担心运行来源不明的混淆 .exe，不知道它在后台做什么 😆
  - 免费使用，并将一直保持免费
